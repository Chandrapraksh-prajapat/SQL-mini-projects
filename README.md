# Library Management SQL Project

This project demonstrates a simple Library Management System built using SQL.

# Database Tables
- Students
- Books
- Issue Records

# Features
- Relational table design using Primary & Foreign Keys
- JOIN operations for multi-table analysis
- GROUP BY & HAVING for aggregation
- NULL handling for pending returns
- Indexing for performance optimization

# Analytical Queries Included
- List all issued books with student names
- Students who issued more than one book
- Pending book returns
- Total books issued per student
- Overdue books

# Skills Used
- SQL
- Joins
- Aggregations
- Database Relationships
- Indexing
